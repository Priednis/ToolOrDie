Let’s talk browser support.

  • C-grade: HTML only. No CSS, no JavaScript.
  • B-grade: HTML with varying support for CSS (perhaps a fixed or single column layout) and feature-tested JavaScript. Usually about media-queryless layout on IE8 and below (Respond.js, ie.scss, etc). I usually don’t do this for personal projects.
  • A-grade: HTML with ‘full’ CSS (responsive layout) and feature-tested JavaScript. Varying support for advanced/non-core functionality.
