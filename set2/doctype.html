The HTML5 DOCTYPE.

<!doctype html>
  • Determines what rendering mode the browser will use (Quirks / Standards).
  • Really easy to forget this, and will cause really strange behavior later.
  • Don’t put HTML comments above it (can trigger Quirks mode in IE)!
document.compatMode == 'BackCompat'; // Quirks Mode
document.compatMode == 'CSS1Compat'; // Standards Mode
