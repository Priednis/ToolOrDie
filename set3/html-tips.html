Use CSS selectors to generate HTML

ul>li*3 becomes 

<ul>
	<li></li>
	<li></li>
	<li></li>
</ul>

Any element name becomes an open and closed tag, but a becomes 

<a href=""></a>
with proper cursor placement.

and many others.

LiveReload

Will refresh your browser for you (without page refresh for CSS)

CodeKit also does this and Yeoman bundles it in as well.

Sublime Text Shortcut

  • ⌘ + / to toggle HTML comments.
  • Ctrl + Shift + W to wrap selection in tags. [1]
  • Ctrl + D to select next occurance of current selection and use multiple cursors (ghetto HTML refactoring).
