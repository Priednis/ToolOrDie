Tool or Die, a Web Developer’s Workflow

Global (correct) fix: add document encoding (charset). Faster: set in HTTP header: Content-Type:text/html; charset=utf-8

Sample .htaccess for Apache

# From HTML5 Boilerplate h5bp.com

# Use UTF-8 encoding for anything served text/plain or text/html
AddDefaultCharset utf-8

# Force UTF-8 for a number of file formats
AddCharset utf-8 .css .js .json
