My Current Bare Minimum for an Editor

  • Quick Project Switching (Ctrl+⌘+P) and File Switching (⌘+P)
  • Syntax Highlighting
  • Easy Plugin Management (Sublime Package Control)
  • Easy global conversion between spaces and tabs
  • Swap Lines without Clipboard, also known as Line Bubbling (swap_line_up or swap_line_down, Alt+Up/Down Arrow)
  • Visually unobtrusive whitespace
  • Duplicate Lines without Clipboard (duplicate_line, Ctrl+⌘+Up/Down Arrow)
  • Some visual integration with my source control (GitGutter)
